# 연말정산 꿀팁 — 딥리서치 분석 결과

> 2025년 귀속분 (2026년 초 신고) 기준

---

## 1. 숨겨진 공제 항목 — 아는 사람만 쓰는 공제

### 1-1. 월세 세액공제 (2025년 대폭 확대)

| 항목 | 내용 |
|---|---|
| 소득 기준 | 총급여 **8,000만 원 이하** (기존 7,000만 원 → 상향) |
| 공제 한도 | 연간 월세액 **1,000만 원** (기존 750만 원 → 상향) |
| 공제율 | 총급여 5,500만 원 이하: **17%** / 5,500만~8,000만 원: **15%** |
| 최대 환급액 | 170만 원 (월세 1,000만 원 × 17%) |

**조건:**
- 본인 및 기본공제대상 가족 모두 무주택자
- 국민주택 규모 이하(85㎡ 이하) 또는 기준시가 4억 원 이하
- 임대차계약서 주소지 = 주민등록 주소지 일치
- 임대인 계좌로 월세 이체 (계좌이체 내역 보관 필수)

### 1-2. 중소기업 청년 소득세 감면 (90% 감면) — 가장 강력

| 항목 | 내용 |
|---|---|
| 감면율 | 청년: **90%** / 고령자·장애인·경력단절여성: **70%** |
| 적용 기간 | 청년: 취업일로부터 **5년간** |
| 연간 감면 한도 | **200만 원** |
| 5년 누적 혜택 | **최대 1,000만 원** |

**청년 조건:**
- 근로계약 체결일 기준 만 15~34세 이하 (군복무 최대 6년 차감, 실질 만 40세)
- 중소기업 규모 요건 충족 (자산 5,000억 미만)
- 신청 기한 놓쳤어도 **경정청구로 5년 소급 적용 가능**

### 1-3. 주택담보대출 이자 소득공제

| 상환 방식 | 공제 한도 |
|---|---|
| 고정금리 + 비거치식 분할상환 | **2,000만 원** |
| 고정금리 or 비거치식 (둘 중 하나) | **1,500만 원** |
| 그 외 (변동금리, 거치식) | **600만 원** |

**조건:** 세대주, 취득 당시 기준시가 6억 원 이하, 상환기간 10년 이상, 주택 명의자 = 대출 명의자

### 1-4. 의료비 세액공제

| 대상 | 공제율 | 한도 |
|---|---|---|
| 일반 부양가족 | 15% | 700만 원 |
| 본인·65세 이상·장애인 | 15% | **한도 없음** |
| 난임시술비 | **30%** | 한도 없음 |
| 미숙아·선천성이상아 | **20%** | 한도 없음 |

**숨은 공제 대상:** 안경·콘택트렌즈(1인 50만 원), 산후조리원(출산 1회당 200만 원), 보청기, 휠체어
**주의:** 실손보험금 수령액은 공제 대상에서 제외. 총급여 3% 초과분부터 공제 시작

### 1-5. 교육비 세액공제

| 대상 | 공제 한도 | 공제율 |
|---|---|---|
| 본인 (대학원, 직업훈련비 포함) | **한도 없음** | 15% |
| 대학생 자녀 | 900만 원/년 | 15% |
| 초중고생 자녀 | 300만 원/년 | 15% |
| 취학전 아동 학원비 | 300만 원/년 | 15% |

**포인트:** 본인 야간대학원 전액 공제, 직무 관련 직업훈련비 공제, 학자금 대출 원리금 상환도 가능

### 1-6. 기부금 세액공제 — 고향사랑기부제

| 기부 금액 | 공제율 | 비고 |
|---|---|---|
| 10만 원 이하 | **100% 전액** | 사실상 무료 기부 |
| 10만 원 초과 ~ 2,000만 원 | 15% | 2025년부터 한도 2,000만 원 확대 |

**핵심:** 10만 원 기부 = 세금 10만 원 환급 + 답례품 3만 원 = 실질 이익 3만 원
**신청:** [고향사랑e음](https://ilovegohyang.go.kr)

### 1-7. 연금저축 + IRP 세액공제

| 총급여 | 공제율 | 900만 원 납입 시 환급액 |
|---|---|---|
| 5,500만 원 이하 | **16.5%** | **148만 5천 원** |
| 5,500만 원 초과 | **13.2%** | **118만 8천 원** |

**추천 전략:** 연금저축 600만 원 + IRP 300만 원 분산 납입

### 1-8. 주택청약저축 소득공제 (2025년 확대)

| 항목 | 내용 |
|---|---|
| 소득 기준 | 총급여 7,000만 원 이하 |
| 대상 | 무주택 세대주 (2025년부터 **배우자도 포함**) |
| 공제 한도 | 납입액 **300만 원** (기존 240만 원 → 상향) |
| 최대 소득공제 | **120만 원** |

## 2. 흔한 실수 TOP 5

### 실수 1. 맞벌이 부부 공제 배분 최적화 미실시
- **원칙:** 소득 높은 쪽에 공제 몰아주기 (단, 의료비는 반대로 소득 낮은 쪽)
- 의료비 기준선 = 총급여의 3% → 소득 낮은 배우자가 기준선이 낮아 더 유리
- 홈택스 "맞벌이 절세 안내" 기능 활용

### 실수 2. 부양가족 중복 공제 / 소득 초과 부양가족 공제
- 기본공제 기준: 연간 소득금액 100만 원 이하 (근로소득만 있으면 총급여 500만 원 이하)
- 형제가 동시에 부모님 공제받으면 한 명 가산세
- 연금소득, 양도소득, 퇴직소득도 소득에 포함

### 실수 3. 신용카드 25% 채우는 순서 오류
| 결제 수단 | 공제율 |
|---|---|
| 신용카드 | **15%** |
| 체크카드·현금영수증 | **30%** |
| 전통시장·대중교통 | **40%** |
| 헬스장·수영장·도서·공연 | **30~40%** |

**전략:** 연봉 25%까지 신용카드(포인트 혜택), 초과분부터 체크카드 전환

### 실수 4. 실손보험금 수령 후 의료비 공제 그대로 신청
- 실손보험금 받은 금액은 반드시 차감 → 미차감 시 가산세

### 실수 5. 월세 공제 주소 불일치 / 대출 명의 불일치
- 임대차계약서 주소 ≠ 주민등록 주소 → 공제 불가
- 주택 명의자 ≠ 대출 명의자 → 공제 불가

## 3. 2025년 새로 바뀐 제도

### 3-1. 결혼세액공제 신설 (2024~2026년 한시)
- 부부 각각 **50만 원**, 합산 최대 **100만 원**
- 생애 1회 (재혼도 적용), 나이 제한 없음

### 3-2. 자녀세액공제 대폭 확대
| 자녀 수 | 기존 | 2025년 |
|---|---|---|
| 첫째 | 15만 원 | **25만 원** |
| 둘째 | 20만 원 | **30만 원** |
| 셋째 이상 | 30만 원 | **40만 원** |

출산공제 중복: 셋째 출생 시 자녀 40만 원 + 출산 70만 원 = **110만 원**

### 3-3. 헬스장·수영장 소득공제 신설 (2025년 7월~)
- 총급여 7,000만 원 이하, 공제율 **30%**, 문화비 통합 한도 300만 원
- 등록된 사업자만 가능

### 3-4. 월세 세액공제 소득기준 확대
- 7,000만 원 → **8,000만 원**, 한도 750만 원 → **1,000만 원**

### 3-5. 주택청약저축 공제 한도 확대
- 240만 원 → **300만 원**, 배우자도 새로 포함

## 5. 영상 포맷 아이디어

### A. 퀴즈 형식 (참여 유도)
- "연봉 3,500만 원인데 환급 0원, 왜일까요?"
- 정답: 신용카드만, IRP 미가입, 중소기업 감면 미신청

### B. Before/After 비교
- A씨(꿀팁 모름): 환급 52만 원
- B씨(꿀팁 적용): IRP + 고향사랑기부 + 헬스장 = 환급 286만 원
- 동일 연봉, 동일 지출 조건

### C. 체크리스트 형식
- "연말정산 전 반드시 확인할 10가지"
- 체크박스 하나씩 체크되는 애니메이션

### D. 실수 사례 → 올바른 방법
- "이렇게 하면 가산세 맞습니다" → "이렇게 바꾸면 환급됩니다"

### E. 직군별 맞춤 가이드
- 중소기업 청년편 / 맞벌이 부부편 / 월세 직장인편
