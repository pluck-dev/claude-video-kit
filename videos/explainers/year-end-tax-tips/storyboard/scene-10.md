## Remotion 컴포넌트 매핑

- **컴포넌트명:** `Scene10Checklist`
- **파일 위치:** `scenes/Scene10-Checklist.tsx`
- **주요 props:**
  ```typescript
  interface Scene10ChecklistProps {
    items: Array<{
      label: string;
      detail: string;
    }>;
    averageRefund: number;    // 68 (만 원)
    maxRefund: string;        // "수백만 원"
  }
  ```
- **사용 애니메이션:**
  - `interpolate(frame, [0, 15], [0, 1])` — 제목 fadeIn
  - `interpolate(frame - 90 - i * 100, [0, 15], [50, 0])` — 각 항목 slideFromRight (i=0~5, 간격 약 3.3s)
  - `interpolate(frame - 90 - i * 100, [0, 15], [0, 1])` — 각 항목 opacity
  - `interpolate(frame - 130 - i * 100, [0, 20], [0, 1])` — 체크 아이콘 drawPath (SVG strokeDashoffset)
  - `spring({ frame: frame - 750 })` — 최종 강조 텍스트 scaleUp
- **체크 아이콘 구현:**
  - SVG `<path>` strokeDasharray + strokeDashoffset interpolate로 그리듯 등장
  - CSS keyframes/animation 사용 금지

- 각 항목 배경 카드: `#1E293B`, border-radius 12px, 체크 완료 시 좌측 테두리 `#10B981` 4px 추가

- "68만 원": `#94A3B8` 취소선 + "수백만 원" 등장으로 대체 효과
- 항목 간격: 각 항목 등장 간격 약 100프레임(3.3s) — 나레이션 속도와 동기화
