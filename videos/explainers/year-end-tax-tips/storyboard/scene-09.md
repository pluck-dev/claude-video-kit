## 나레이션

> 신용카드 쓰는 순서도 중요합니다.
>
> 연봉의 25%까지는 신용카드로 쓰세요.
> 포인트 혜택이 있으니까요.
> 25% 넘어가는 순간부터는 체크카드로 전환하세요.
> 체크카드 공제율이 30%로 더 높습니다.
> 전통시장이나 대중교통은 40%까지 올라가요.
>
> 그리고 자녀 있으신 분들, 올해 공제 확 늘었습니다.
> 첫째 25만 원, 둘째 30만 원, 셋째 40만 원.
> 셋째 낳으셨으면요? 출산공제 70만 원이 추가됩니다.
> 합치면 110만 원이에요.
>
> 본인이 대학원 다니시는 분도 있죠?
> 본인 교육비는 한도가 없습니다.
> 전액 15% 공제예요. 직업훈련비도 포함됩니다.

## Remotion 컴포넌트 매핑

- **컴포넌트명:** `Scene09CardStrategy`
- **파일 위치:** `scenes/Scene09-CardStrategy.tsx`
- **주요 props:**
  ```typescript
  interface Scene09CardStrategyProps {
    threshold: number;       // 25 (%)
    creditRate: number;      // 15 (%)
    checkRate: number;       // 30 (%)
    marketRate: number;      // 40 (%)
    children: Array<{ order: string; amount: number }>;
    birthBonus: number;      // 70 (만 원)
    educationRate: number;   // 15 (%)
  }
  ```
- **사용 애니메이션:**
  - `interpolate(frame, [90, 210], [0, 1])` — 신용카드 바 grow
  - `interpolate(frame, [120, 240], [0, 1])` — 체크카드 바 grow (지연)
  - `interpolate(frame, [150, 270], [0, 1])` — 시장/교통 바 grow (지연)
  - `interpolate(frame - 540 - i * 15, [0, 20], [30, 0])` — 자녀 테이블 행 순차 slideFromBottom (i=0,1,2,3)
  - `spring({ frame: frame - 900 })` — 교육비 카드 scaleUp
