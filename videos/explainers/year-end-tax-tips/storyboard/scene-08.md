## 나레이션

> 이제 실수 이야기 해볼게요.
> 이거 잘못하면 오히려 가산세 맞습니다.
>
> 실수 1번, 맞벌이 부부 공제 배분.
> 소득이 높은 쪽에 공제를 몰아주는 게 맞아요.
> 근데 의료비만 반대입니다.
> 의료비는 소득 낮은 배우자에게 몰아주세요.
> 이유는 총급여의 3% 초과분부터 공제되기 때문입니다.
> 소득이 낮을수록 기준선이 낮아서 더 많이 공제돼요.
> 홈택스에 "맞벌이 절세 안내" 기능 있으니 꼭 써보세요.
>
> 실수 2번, 부양가족 중복 공제.
> 부모님 공제를 형제 둘이서 같이 받으면 한 명은 가산세입니다.
> 연간 소득금액 100만 원 이하인 분만 공제 대상이에요.
> 근로소득만 있으면 총급여 500만 원 이하여야 합니다.
>
> 실수 3번, 실손보험금 미차감.
> 의료비 쓰고 실손보험금 받았으면, 그 금액은 빼야 해요.
> 안 빼면 가산세 나옵니다.

## Remotion 컴포넌트 매핑

- **컴포넌트명:** `Scene08Mistakes`
- **파일 위치:** `scenes/Scene08-Mistakes.tsx`
- **주요 props:**
  ```typescript
  interface Scene08MistakesProps {
    mistakes: Array<{
      number: number;
      title: string;
      description: string[];
      tip?: string;
    }>;
  }
  ```
- **사용 애니메이션:**
  - `interpolate(frame, [0, 15], [0, 1])` — 경고 배너 fadeIn
  - `interpolate(frame % 10, [0, 5, 10], [0, 3, 0])` — 경고 배너 shake (translateX)
  - `interpolate(frame, [120, 145], [-100, 0])` — 실수 1 카드 slideFromLeft
  - `interpolate(frame, [660, 685], [-100, 0])` — 실수 2 카드 slideFromLeft
  - `interpolate(frame, [1200, 1225], [-100, 0])` — 실수 3 카드 slideFromLeft
  - `interpolate(frame, [120, 135], [0, 1])` — 실수 1 카드 opacity
  - X→O 전환: `interpolate(frame, [180, 210], [0, 1])` — 체크 아이콘 opacity

## 특이사항

- 씬 전체 톤: 빨간색 계열 강조 — 경고/주의 분위기
- 경고 배너: `#EF4444` 배경, `#F8FAFC` 텍스트, 좌우 전체 너비
- 배너 shake 애니메이션: translateX를 `interpolate(frame % 10, [0,2,4,6,8,10], [0,3,-3,3,-3,0])` 로 구현 (CSS keyframes 금지)
- 각 실수 카드: `#1E293B` 배경, `#EF4444` 좌측 4px 세로 테두리 (border-left)
- "✗" 아이콘: `#EF4444` 색상 → 올바른 방법 표시 시 "✓" `#10B981` 색상으로 전환
- X→O 전환: opacity interpolate로 X 아이콘 fade out 후 O 아이콘 fade in
- "가산세" 단어: 모든 위치에서 `#EF4444` + Bold 강조
- 홈택스 팁: `#2563EB` 배경의 작은 인포 배지
