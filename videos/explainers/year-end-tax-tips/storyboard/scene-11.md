## Remotion 컴포넌트 매핑

- **컴포넌트명:** `Scene11Outro`
- **파일 위치:** `scenes/Scene11-Outro.tsx`
- **주요 props:**
  ```typescript
  interface Scene11OutroProps {
    channelName: string;
    nextVideoTitle: string;   // "프리랜서 종합소득세 신고 꿀팁"
    nextVideoHook: string;    // "모르면 수십만 원 날린다!"
  }
  ```
- **사용 애니메이션:**
  - `interpolate(frame, [0, 20], [0, 1])` — 배경 + 대제목 fadeIn
  - `interpolate(frame, [90, 110], [30, 0])` — 로고 + 부제 slideFromBottom
  - `interpolate(frame, [90, 110], [0, 1])` — 로고 opacity
  - `spring({ frame: frame - 240, config: { mass: 0.6 } })` — 구독 버튼 scaleUp
  - `interpolate(frame, [420, 445], [100, 0])` — 예고 카드 slideFromRight
  - `interpolate(frame, [420, 445], [0, 1])` — 예고 카드 opacity
  - `interpolate(frame, [570, 600], [1, 0])` — 전체 fadeOut

## 특이사항

- 배경: `#0F172A` 단색 + 미세한 그라디언트 오버레이 (`#0F172A` → `#1E293B`) 방사형
- 채널 로고: 중앙 상단 배치, 크기 120px × 120px
- 구독 버튼: `#EF4444` (유튜브 빨강) 배경, `#F8FAFC` 텍스트, border-radius 8px
- 구독 버튼 hover 효과 없음 (정적 영상) — pulse 애니메이션으로 시선 유도
  - `interpolate(frame % 45, [0, 22, 45], [1, 1.04, 1])` — 1.5초 주기 pulse
- 다음 영상 예고 카드: `#1E293B` 배경, `#2563EB` 상단 테두리 4px, border-radius 16px
- 예고 카드 내 "수십만 원": `#EF4444` Bold 강조
- 최종 fadeOut: 마지막 30프레임(1초) 동안 전체 opacity 1→0
