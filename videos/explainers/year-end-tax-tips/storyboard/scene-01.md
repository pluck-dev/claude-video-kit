## Remotion 컴포넌트 매핑

- **컴포넌트명:** `Scene01Hook`
- **파일 위치:** `scenes/Scene01-Hook.tsx`
- **주요 props:**
  ```typescript
  interface Scene01HookProps {
    personA: { name: string; amount: number };   // { name: "A씨", amount: 52 }
    personB: { name: string; amount: number };   // { name: "B씨", amount: 286 }
    salary: number;                              // 3500 (만 원)
  }
  ```
- **사용 애니메이션:**
  - `interpolate(frame, [0, 15], [0, 1])` — fadeIn 배경 텍스트
  - `interpolate(frame, [30, 60], [0, 1920/2])` — 왼쪽 패널 슬라이드
  - `spring({ frame: frame - 90, fps, config: { mass: 0.5 } })` — 숫자 카운트업
  - `interpolate(frame, [360, 380], [0, 1])` — 질문 텍스트 fadeIn
